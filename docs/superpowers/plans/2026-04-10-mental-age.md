# 心理年龄测试 Skill 实现计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 创建搞笑玩梗风格的心理年龄测试 skill，20 题 × 4 选项，输出具体岁数 + 8 个人生阶段标签。

**Tech Stack:** Markdown only, no code dependencies.

**Design Spec:** `docs/superpowers/specs/2026-04-10-mental-age-design.md`

### Task 6: 输出报告模板 + 快速参考 + 常见错误 + 总结

**Files:**
- Modify: `skills/mental-age/SKILL.md`（在详细解读之后追加）

- [ ] **Step 1: 编写输出报告模板**

模板前加使用说明：
- 统计 20 题总分，除以 20 得平均分
- 平均分 = 心理年龄
- 查阶段对照表确定人生阶段标签

输出模板内容：
- 标题区（心理年龄岁数 + 阶段标签 + emoji + 一句话）
- 各题得分明细（20 个分值）
- 平均分计算结果
- 阶段解读（从对应阶段详细解读复制）
- 和同龄人的差距（实际年龄 vs 心理年龄）
- 名场面（从对应阶段复制）
- 建议（从对应阶段复制）

- [ ] **Step 2: 编写快速参考表**

| 需要什么 | 去哪找 |
|----------|--------|
| 给某人做心理年龄测试 | 使用"心理年龄测试题"部分，逐题出题 |
| 查看某个阶段的解读 | 查看"人生阶段详细解读"对应阶段 |
| 计算心理年龄 | 统计20题总分÷20，查阶段对照表 |
| 生成测试报告 | 使用"输出报告模板" |
| 只做了10题能出结果吗 | 不建议——20题取平均才准确 |

- [ ] **Step 3: 编写常见错误 + 总结**

常见错误（6条）：

| 错误做法 | 正确做法 |
|----------|----------|
| 把心理年龄当真实年龄 | 这只是趣味测试，不代表心智成熟度 |
| 觉得心理年龄越小越好 | 每个阶段都有独特价值 |
| 心理年龄大就是"老" | 心理年龄大可能意味着通透和智慧 |
| 只做几道题就出结果 | 20题取平均才准确 |
| 重复做到满意为止 | 第一反应最真实，不要"优化"答案 |
| 拿结果嘲笑别人 | 每个人的节奏不同，没有优劣 |

总结（3-5句）：
- 心理年龄反映的是行为模式偏好，不是真实水平
- 不管你是幼儿园小霸王还是退休大爷，开心最重要
- 本测试纯属娱乐，没有任何科学依据
- 分享给朋友一起测，看看谁最"老"

```bash

git commit -m "feat(mental-age): 添加输出模板、快速参考、常见错误和总结"
```
