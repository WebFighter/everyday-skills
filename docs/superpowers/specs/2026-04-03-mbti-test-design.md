# MBTI 人格测试 Skill 设计文档

## 目标

创建一个完整的 MBTI 人格测试 skill，包含分维度测试、16种类型详细解读、双类型匹配分析三大功能。单文件实现，中英混合输出。

## 功能范围

### 功能一：分维度测试

按 E/I、S/N、T/F、J/P 四个维度分组测试，每组5道二选一题。

- 每题给出两个选项（A/B），对应维度的两端
- 每个维度独立计分：多选某端则判定为该偏好
- 2:3 或 3:2 → 轻微偏好；4:1 或 5:0 → 强烈偏好
- 四个维度组合出16种类型之一（如 INTJ、ENFP）
- 测试结果标注偏好强度（如 I(强烈) N(轻微) T(强烈) J(轻微)）

测试题目设计原则：
- 避免明显的"正确答案"
- 场景化而非抽象提问（如"周末你更倾向于..."而非"你是内向的吗"）
- 每道题的两个选项应该看起来都合理

### 功能二：16种类型解读

为每种类型提供以下信息：

1. **类型代码 + 中文名** — 如 INTJ 建筑师
2. **核心特质** — 2-3句话概括
3. **性格优势** — 3-4个关键词 + 一句话说明
4. **潜在盲区** — 2-3个需要注意的方面
5. **职业方向** — 适合的职业领域
6. **人际关系** — 社交风格和亲密关系特点

16种类型速查表（代码+中文名+一句话概括）放在解读部分之前，方便快速查阅。

### 功能三：双类型匹配分析

输入两个MBTI类型，输出匹配分析报告。

匹配分析基于**维度互补规则**，不穷举256种组合：

| 维度关系 | 含义 | 影响 |
|----------|------|------|
| 完全相同 | 高度理解，但可能缺乏互补 | 沟通顺畅但视角单一 |
| 部分互补 | 有共同点也有差异 | 平衡性最好 |
| 完全相反 | 互补性强但冲突风险高 | 成长空间大但需磨合 |

输出内容：
1. **匹配等级** — 高度契合 / 良好互补 / 需要磨合 / 差异较大
2. **核心动态** — 这对组合的典型相处模式
3. **沟通建议** — 如何有效沟通
4. **潜在冲突点** — 最容易产生分歧的领域
5. **成长建议** — 如何利用差异互相成长

## 文件结构

```
skills/
  mbti-test/
    SKILL.md    # 单文件，包含所有内容
```

## 输出语言约定

- MBTI 类型代码：英文（INTJ、ENFP 等）
- 类型中文名：中文（"建筑师"、"竞选者" 等）
- 所有解读、建议、分析：中文
- 维度名称：中英混合（E/I 外向/内向）

## 不包含的功能

- 不包含测试结果的可视化图表
- 不包含与其他心理测评（如九型人格、大五）的交叉分析
- 不包含结合 xiangqin-matching skill 的综合匹配
- 不提供测试题目随机化功能

## SKILL.md 内部结构

1. YAML 前置信息（name: mbti-test, description）
2. 概述 + 何时使用/不适用
3. 分维度测试（4组题目 + 计分规则）
4. 16种类型速查表
5. 16种类型详细解读
6. 类型匹配分析框架 + 维度互补规则
7. 输出报告模板
8. 快速参考 + 常见错误
