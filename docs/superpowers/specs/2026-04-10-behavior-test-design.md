# 行为测试 Skill 设计文档

## 目标

创建一个纯行为驱动的趣味测试 skill，通过 36 道场景选择题测量 12 个行为维度，输出多维度评分 + 16 种核心行为类型归档。风格为玩梗搞笑，但基于真实可观察行为。

**核心理念：只问"你怎么做"，不问"你怎么想"。**

## 功能范围

### 功能一：12 维度行为测试

36 道场景选择题，每题 A/B 两个行为选项。12 个维度各 3 道题。

| # | 维度名 | 测量的行为 | A端（搞笑标签） | B端（搞笑标签） |
|---|--------|-----------|----------------|----------------|
| 1 | 行动速度 | 想到→做到的时间差 | ⚡ 闪现侠 | 🐢 慢半拍 |
| 2 | 社交引擎 | 社交时主动还是被动 | 🔊 人形扩音器 | 🔇 隐形人 |
| 3 | 冒险指数 | 面对新事物敢不敢试 | 🎲 赌狗本狗 | 🛡️ 安全堡垒 |
| 4 | 混乱容忍 | 生活环境的秩序感 | 📋 强迫症晚期 | 🌀 混沌大师 |
| 5 | 信息深度 | 获取信息深浅 | 🔍 考据狂魔 | 📱 刷屏达人 |
| 6 | 压力反应 | 压力下干什么 | 💥 暴走模式 | 🛋️ 躺平仙人 |
| 7 | 消费模式 | 实际花钱行为 | 💸 撒币大师 | 🐷 铁公鸡 |
| 8 | 时间观念 | 守时还是迟到 | ⏰ 提前到狂魔 | 🕐 迟到惯犯 |
| 9 | 表达方式 | 说话直接还是绕弯 | 🎯 直球选手 | 🌀 弯弯绕绕 |
| 10 | 计划性 | 提前安排还是随性 | 📅 计划通 | 🎲 走一步算一步 |
| 11 | 情绪外露 | 情绪是否写在脸上 | 😭 行走的表情包 | 🗿 扑克脸 |
| 12 | 竞争意识 | 面对竞争时卷不卷 | 🔥 卷王本王 | 🧘 佛系青年 |

### 计分规则

每维度 3 题：
- 3:0 → 极端偏好（★★★），该维度标签**加粗**
- 2:1 → 中度偏好（★★☆），该维度正常标签

### 功能二：两层分类体系

#### 第一层：4 个元维度（决定核心类型）

| 元维度 | 包含维度 | A极（元标签） | B极（元标签） |
|--------|---------|--------------|--------------|
| 执行模式 | 行动速度 + 计划性 + 时间观念 | ⚡ 闪电执行者 | 🐢 慢速巡航者 |
| 社交风格 | 社交引擎 + 表达方式 + 情绪外露 | 🎭 社交核弹 | 🫥 社交隐身 |
| 探索态度 | 冒险指数 + 信息深度 + 消费模式 | 🚀 莽撞探险家 | 🏠 居家守门员 |
| 应对策略 | 压力反应 + 混乱容忍 + 竞争意识 | ⚔️ 硬刚战士 | 🧘 佛系仙人 |

元维度评分：组内 3 个维度取多数极性（2-3 个同极→该元维度定极）。

4 个元维度 × 2 极 = **16 种核心行为类型**。

#### 第二层：12 维度个体画像

展示 12 个维度的完整评分，标出极端维度（最强偏好），给出个性化行为标签。

## 题目设计原则

1. **只问行为，不问想法** — "你会怎么做" 而非 "你觉得你应该怎么做"
2. **场景具体、接地气** — 吃饭、网购、社交、上班、游戏等日常
3. **两个选项都合理** — 不暗示某个选项更好
4. **搞笑但不冒犯** — 有梗但不会让任何人选完后不舒服

## 文件结构

```
skills/
  behavior-test/
    SKILL.md    # 单文件，包含：
                # 1. YAML 前置信息 + 概述
                # 2. 12维度测试题（36题，每维度3题）
                # 3. 计分规则
                # 4. 16种类型速查表
                # 5. 16种类型详细解读
                # 6. 输出报告模板
                # 7. 快速参考 + 常见错误
```

## 输出语言约定

- 类型名称：中文（"人形龙卷风"、"考拉成精"等）
- 维度名称：中文 + emoji
- 所有解读、建议、分析：中文
- 行为标签：`#标签` 格式

## 不包含的功能

- 不包含测试结果的可视化图表
- 不包含与 MBTI 或其他心理测评的交叉分析
- 不提供测试题目随机化功能
- 不包含在线测试网页（仅作为 skill 文本）
- 不包含科学性声明（明确标注为趣味测试）

## 与现有 skill 的关系

- 独立于 mbti-test skill，不共享数据或依赖
- 可与 xiangqin-matching skill 配合使用（行为类型作为匹配参考维度之一），但不在本 skill 内实现
