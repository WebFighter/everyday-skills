# 心理年龄测试 Skill 设计文档

## 目标

创建一个搞笑玩梗风格的心理年龄测试 skill，通过 20 道场景选择题（每题 4 选项）测量心理年龄，输出具体岁数 + 人生阶段标签。风格参考 behavior-test skill。

**核心理念：通过日常行为看透你的"真实年龄"。**

## 功能范围

### 功能一：心理年龄测试

20 道场景选择题，每题 A/B/C/D 四个选项。4 个选项从"最年轻行为"到"最成熟行为"排列，每个选项对应一个心理年龄分值。

**评分分值范围：**
- A → 天真童趣型行为（分值 5-15）
- B → 年轻冲劲型行为（分值 15-25）
- C → 成熟稳重型行为（分值 25-45）
- D → 佛系通透型行为（分值 45-70）

每题分值不完全相同（增加精度），20 道题总分取平均 → 心理年龄岁数。

### 场景覆盖

20 道题覆盖 10 个生活场景，每个 2 题：

| # | 场景 | 示例话题 |
|---|------|---------|
| 1-2 | 作息起床 | 早上醒来第一件事、晚上几点睡 |
| 3-4 | 消费花钱 | 逛街看到喜欢的东西、月底钱包状态 |
| 5-6 | 社交方式 | 微信回消息速度、聚会的角色 |
| 7-8 | 情绪管理 | 被人批评后、心情不好的应对 |
| 9-10 | 娱乐偏好 | 刷到什么内容会停下来、周末做什么 |
| 11-12 | 压力应对 | 面对截止日期、遇到困难时 |
| 13-14 | 人际关系 | 和朋友吵架后、被放鸽子 |
| 15-16 | 沟通表达 | 收到不喜欢的礼物、反驳别人 |
| 17-18 | 工作学习 | 开会/上课的状态、对待规则 |
| 19-20 | 生活态度 | 对未来的想法、人生信条 |

### 计分规则

- 统计 20 道题的总分，除以 20，得到平均分
- 平均分 = 心理年龄（岁）
- 根据平均分查表，确定人生阶段标签

|-----------|---------|-------|--------|
| 0-12 | 幼儿园小霸王 | 🧒 | 世界是你的游乐场 |
| 13-17 | 中二少年 | 🤘 | 你觉得全世界都不懂你 |
| 18-25 | 永远的大学生 | 🎓 | 熬夜打游戏，明天再说 |

| 34-42 | 人间清醒 | 🧠 | 看透了一半，另一半懒得看了 |
| 43-52 | 活得通透 | ☕ | 别人说啥你都说"嗯，好的" |

每种阶段包含：
1. **阶段标签 + emoji**
2. **一句话描述**
3. **行为画像** — 2-3 段描述这个阶段的人的日常行为模式
4. **名场面** — 一个典型场景对话
5. **给你的建议** — 1-2 句搞笑建议

## 题目设计原则

1. **4 选项从年轻到成熟排列** — A 最年轻、D 最成熟
2. **每个选项都有分值** — 分值标注在选项后面
3. **场景具体、接地气** — 日常生活场景
4. **搞笑但不冒犯** — 有梗但不会让任何人不舒服
5. **分值有梯度** — 每题的 4 个分值拉开差距

## 输出语言约定

- 阶段标签：中文（"幼儿园小霸王"、"老江湖"等）
- 所有解读、建议：中文
- 分值：数字（如"6分"、"32分"）

## 不包含的功能

- 不包含与实际年龄的精确对比算法
- 不包含心理学/发展心理学的科学依据
- 不包含测试题目随机化功能
- 不包含在线测试网页
- 明确标注为趣味测试，无科学依据

## 与现有 skill 的关系

- 独立于 behavior-test skill，不共享数据或依赖
- 风格和结构参考 behavior-test（搞笑玩梗、场景选择题），但测量内容不同
