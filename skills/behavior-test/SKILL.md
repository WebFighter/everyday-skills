# 行为测试

## 概述

一个纯行为驱动的趣味测试工具。**核心理念："只问你怎么做，不问你怎么想。"**

通过 36 道场景选择题测量 12 个行为维度，再归入 4 个元维度，最终输出你的多维度评分和 16 种核心行为类型归属。风格玩梗搞笑，但基于真实可观察行为——所有题目都只问你过去的实际行动，而不是你的想法、态度或价值观。

## 何时使用

- 有人想快速了解自己的行为模式和行为风格
- 朋友之间想互相了解对方的日常做事习惯
- 团队活动中需要一个轻松有趣的行为测试破冰
- 想知道自己和另一个人在行为层面合不合拍
- 纯粹好奇自己属于 16 种行为类型中的哪一种

## 何时不适用

- 临床心理诊断或心理健康评估（本测试为趣味测试，非诊断工具）
- 招聘筛选或人事决策（不应仅凭行为测试做决策）
- 替代专业心理咨询或性格分析
- 对行为结果过度解读或贴标签（行为偏好无好坏之分）

## 两层分类体系

### 第一层：4 个元维度（决定核心类型）

每 3 个行为维度归入 1 个元维度，元维度的极性由组内多数维度的极性决定：

| 元维度 | 包含维度 | A极（元标签） | B极（元标签） |
|--------|---------|--------------|--------------|
| 执行模式 | 行动速度 + 计划性 + 时间观念 | ⚡ 闪电执行者 | 🐢 慢速巡航者 |
| 社交风格 | 社交引擎 + 表达方式 + 情绪外露 | 🎭 社交核弹 | 🫥 社交隐身 |
| 探索态度 | 冒险指数 + 信息深度 + 消费模式 | 🚀 莽撞探险家 | 🏠 居家守门员 |
| 应对策略 | 压力反应 + 混乱容忍 + 竞争意识 | ⚔️ 硬刚战士 | 🧘 佛系仙人 |

**元维度判定规则：** 组内 3 个维度取多数极性——2-3 个维度同极，则该元维度定为该极。

### 第二层：12 维度个体画像

在确定核心类型之外，展示 12 个维度的完整评分，标出极端维度（★★★ 最强偏好），给出个性化行为标签组合。

## 计分规则

每维度 3 题，统计 A/B 选项数量：

- **3:0** → 极端偏好（★★★），该维度标签**加粗**显示
- **2:1** → 中度偏好（★★☆），该维度正常显示

**元维度评分：** 组内 3 个维度取多数极性（2-3 个同极 → 该元维度定极）。

4 个元维度 x 2 极 = **16 种核心行为类型**。

## 12 维度行为测试

### 维度一：行动速度（⚡ 闪现侠 / 🐢 慢半拍）

**1. 你刷手机看到一个很想买的东西，你实际上会怎么做：**
- A) 手比脑子快，还没看完详情页就已经下单了，收货后才想起来自己买了啥
- B) 先加购物车，然后退出APP，过三天再打开看一眼，又退出去，如此循环至少三轮

**2. 周五晚上八点，朋友突然喊你出去吃火锅，你实际会怎么做：**
- A) 秒回"地址"，同时已经在穿鞋了，还顺手查了一下哪家火锅店不用排队
- B) 先问"还有谁"，再问"几点结束"，然后思考半小时要不要洗头，最后回复"下次一定"

**3. 你突然想起一件事要跟同事说，你实际上会怎么做：**
- A) 想到的瞬间就已经打开对话框开始打字了，手速和思路同步起飞
- B) 心里记了一下，想着"等下说"，然后该干啥干啥，三天后突然想起来还没说

> 计分：A = 闪现侠 ⚡，B = 慢半拍 🐢

### 维度二：社交引擎（🔊 人形扩音器 / 🔇 隐形人）

**1. 你第一天到一个新公司/新班级/新社群，你实际上会怎么做：**
- A) 主动挨个自我介绍，中午就拉着一桌人吃饭，下班前已经建好了微信群
- B) 找个角落坐下，默默观察谁看起来比较安全，等别人先来找你说话

**2. 你在一个20人的微信群里，大家正在热烈讨论周末去哪玩，你实际会怎么做：**
- A) 直接甩三个方案出来，附上价格对比和路线图，顺便@所有人投票
- B) 默默翻完所有聊天记录，心里有了答案，但就是不发出来，等别人提到你想去的那个再点个赞

**3. 朋友生日聚会，现场有8个人你都不认识，你实际会怎么做：**
- A) 端着饮料就开始"嗨我是谁谁谁你呢你做什么的哇好巧"，半小时后全场都认识了
- B) 紧跟喊你来的那个朋友，他去哪你去哪，他去上厕所你就低头玩手机假装很忙

> 计分：A = 人形扩音器 🔊，B = 隐形人 🔇

### 维度三：冒险指数（🎲 赌狗本狗 / 🛡️ 安全堡垒）

**1. 你路过一家新开的餐厅，招牌上写着"地狱级魔鬼辣挑战——吃完免单"，你实际会怎么做：**
- A) 直接走进去坐下来，心想"免费吃饭还能挑战自我，双赢啊"
- B) 掏出手机搜了一下这家店的评价，看到有人说"辣到怀疑人生"，默默路过，去隔壁吃了碗面

**2. 旅行到了一个陌生的城市，离飞机起飞还有6个小时，你实际会怎么做：**
- A) 把行李寄存在车站，随便跳上一辆公交车坐到终点站，看看会到什么神奇的地方
- B) 提前查好机场到市区的交通时间，留出2小时buffer，在机场咖啡厅坐等登机

**3. 朋友发来一个链接："周末一起去学冲浪/跳伞/开飞机，报名费刚打折"，你实际会怎么做：**
- A) 秒回"报名了"然后开始收拾行李，至于自己会不会游泳/恐高/晕车——到时候再说
- B) 先问"有教练吗""安全吗""有没有保险""能不能退""之前出过事吗"，问完报名已经截止了

> 计分：A = 赌狗本狗 🎲，B = 安全堡垒 🛡️

### 维度四：混乱容忍（📋 强迫症晚期 / 🌀 混沌大师）

**1. 你的电脑桌面现在是什么状态：**
- A) 文件夹按项目分类，子文件夹按日期命名，桌面最多留3个快捷方式，回收站定期清空
- B) 整个桌面铺满了"新建文件夹""新建文件夹(2)""最终版""最终版真的""打死不改版"，找个文件得用搜索

**2. 你吃完一顿外卖，桌上的垃圾和餐盒你实际会怎么处理：**
- A) 吃完立刻收拾干净，桌面恢复原样，垃圾分类丢好，筷子摆回筷子筒
- B) 先放着吧，等下次再吃外卖的时候一起收拾……然后桌上就摞了三四个外卖盒

**3. 你准备出门上班，钥匙手机钱包这三样东西，你实际是怎么管理的：**
- A) 每样东西都有固定位置，出门前摸三下确认（钥匙在门口挂钩、手机在右手口袋、钱包在包里固定夹层）
- B) 每天出门都像寻宝游戏——钥匙在昨天穿的裤子里，手机夹在沙发缝中，钱包？什么是钱包，你只用手机支付

> 计分：A = 强迫症晚期 📋，B = 混沌大师 🌀

### 维度五：信息深度（🔍 考据狂魔 / 📱 刷屏达人）

**1. 你准备买一部新手机，你实际会怎么做功课：**
- A) 先看十篇深度评测，对比五个品牌的参数表，研究CPU跑分、散热方案、屏幕色域，最后写了一个Excel
- B) 刷到某个博主说"这部手机真香"就下单了，或者直接买跟上一个同品牌的，懒得比

**2. 你追完一部特别喜欢的剧/看完一本特别好的书，你实际会怎么做：**
- A) 去查导演/作者的其他作品，翻背景设定资料，看采访视频，逛论坛帖子，把相关人物关系图画出来
- B) 感叹一句"真好看"，然后刷到下一个推荐就去看新的了

**3. 你突然对一个新话题产生兴趣（比如"量子力学""咖啡烘焙""投资理财"），你实际会怎么做：**
- A) 先买两本书，订阅三个播客，加入一个社群，做笔记，列出学习路线图，准备系统攻克
- B) 看了一个5分钟的科普短视频，觉得"哦原来是这样"，然后就没了，下一个话题

> 计分：A = 考据狂魔 🔍，B = 刷屏达人 📱

### 维度六：压力反应（💥 暴走模式 / 🛋️ 躺平仙人）

**1. 明天中午12点要交一份重要报告，现在是前一天晚上10点，你刚发现还有一个大模块没做完，你实际会怎么做：**
- A) 猛灌一杯咖啡，撸起袖子直接开干，干到凌晨四点搞完，发邮件，然后精神抖擞地睡不着了
- B) 看了一眼时间，叹口气，设了个早上6点的闹钟，然后躺下先睡了——万一明天做不完就做不完吧

**2. 你和同事/朋友发生了严重分歧，对方当众质疑你的方案，你实际会怎么做：**
- A) 当场开始逐条反驳，掏出数据和截图，用逻辑碾压回去，事后还意难平又发了三条消息补充论据
- B) 嘴上说"你说得对我再想想"，内心已经不想争了，回家躺在床上还在想但绝对不会主动再去提

**3. 倒霉的一天：出门被雨淋、地铁坐过站、手机摔了一下屏幕裂了，你实际会怎么做：**
- A) 站在雨里怒买一个手机壳，然后杀去最近的维修店，路上顺手把今天所有倒霉事发朋友圈宣泄，再给自己点杯奶茶化悲愤为行动力
- B) 找个屋檐站着淋会雨，接受今天的命运，心里默默想"算了今天就到这吧"，回家躺在床上刷短视频麻痹自己

> 计分：A = 暴走模式 💥，B = 躺平仙人 🛋️

### 维度七：消费模式（💸 撒币大师 / 🐷 铁公鸡）

**1. 你逛商场路过一家店，看到一件衣服觉得还不错但没在计划内，你实际会怎么做：**
- A) 摸了一下面料就觉得"这就是命"，直接试穿然后结账，回家才发现衣柜里已经有三件差不多的了
- B) 拍了个照片想回来网上比比价，然后打开手机搜同款，比完价又看了看评价，最后觉得"其实也没有那么好看"，啥也没买

**2. 双十一/618大促，你的购物车实际上会变成什么样：**
- A) 从10月就开始往购物车里塞东西，到了当天一顿操作猛如虎，凌晨两点还在抢最后一件满减凑单的商品，第二天收到十几个快递
- B) 提前算了三遍满减公式，列了一张精确到小数点的清单，最后只买了卫生纸和洗衣液，觉得自己又省了一个亿

**3. 你中午跟同事一起吃饭，到了一家人均80的餐厅，你实际会怎么做：**
- A) 看菜单"这个想吃那个也想尝"，点了五个菜两个人吃，吃不完打包还说"人生嘛该吃吃该喝喝"
- B) 先扫一眼菜单价格，默默在心里算了一下人均，然后点了最便宜的那个套餐，还跟同事说"我最近在减肥吃不了太多"

> 计分：A = 撒币大师 💸，B = 铁公鸡 🐷

### 维度八：时间观念（⏰ 提前到狂魔 / 🕐 迟到惯犯）

**1. 你跟朋友约了下午两点在咖啡厅见面，你实际上几点到：**
- A) 一点四十就到了，先找了个好位置坐下，点了杯咖啡刷了二十分钟手机，然后开始琢磨朋友怎么还不来
- B) 一点五十才从家里出门，路上还顺路买了杯奶茶，到的时候两点二十，笑着说"堵车了堵车了"但其实根本没有车

**2. 你要去赶一趟早上八点的飞机/高铁，你实际会怎么安排出门时间：**
- A) 闹钟设在凌晨五点，五点半出门，六点到达机场/车站，在候机厅吃了顿早餐还逛了免税店，觉得时间刚刚好
- B) 闹钟响了三次才起来，疯狂收拾冲出门，一路狂奔，在关门的最后一秒挤上去，坐下才开始喘气

**3. 朋友跟你说"再给我10分钟，我马上到"，你的实际反应是：**
- A) 心里默数十分钟，第十分钟零一秒就开始发消息问"到哪了"，并且已经规划好了如果超过十五分钟就自己去吃
- B) 想都不用想，这个"10分钟"至少得打三折——自己先点杯喝的，再刷两集综艺，朋友来了再说

> 计分：A = 提前到狂魔 ⏰，B = 迟到惯犯 🕐

### 维度九：表达方式（🎯 直球选手 / 🌀 弯弯绕绕）

- B) "嗯……这家啊……也不是不行……就是上次我去的时候感觉一般般……你有没有其他想吃的呀？"，说了半天也没说出"不好吃"三个字

2. **判定 4 个元维度极性：** 组内 3 个维度取多数极性（2-3 个同极 → 该元维度定极）。例如"行动速度 ⚡A + 计划性 📅A + 时间观念 ⏰A"= ⚡闪电执行者；"行动速度 ⚡A + 计划性 📅B + 时间观念 ⏰B"= 🐢慢速巡航者。
3. **查速查表定类型：** 根据 4 个元维度极性组合，在"16 种行为类型速查表"中找到对应编号和类型名称。
4. **生成极端行为标签：** 从所有 ★★★ 的维度中，取对应的搞笑标签（如"闪现侠 ⚡""人形扩音器 🔊"等），组合为极端行为标签。

## 常见错误

| 错误做法 | 正确做法 |
|----------|----------|
| 把结果当成科学测评 | 本测试是趣味行为观察，无科学依据 |
| 用结果给人贴标签 | 每个人都是独特的，这只是行为倾向参考 |
| 认为某个类型"更好" | 没有优劣之分，每种行为模式都有适用场景 |
| 测试一次就定终身 | 行为模式会随环境、经历变化 |
| 忽略偏好强度只看类型名 | ★★★ 和 ★★☆ 的行为强度差异很大 |
| 在严肃场景中引用测试结果 | 请勿用于招聘、考核、婚恋决策 |

## 总结

行为测试是认识自己日常行为模式的起点，不是终点——它帮你看到自己的行为倾向，但绝不应该成为框住自己的标签。重点关注极端维度（★★★）的行为模式，那里藏着你的行为DNA。善用搭配建议理解不同行为风格的互动方式，你会发现那些"跟你合不来"的人，可能只是行为节奏不同。本测试纯属娱乐，没有任何科学依据，认真你就输了。分享给朋友一起测，看看你们的行为化学反应——这才是这个测试最正确的打开方式。
