# MBTI 人格测试与类型分析

## 概述

基于 Myers-Briggs Type Indicator 的完整人格测试和分析工具。包含分维度测试、16种人格类型详细解读、以及双类型匹配度分析三大功能。

**核心框架：** 四个维度（E/I 外向/内向、S/N 感觉/直觉、T/F 思考/情感、J/P 判断/知觉）组合出16种人格类型，每种类型有独特的认知功能栈和行为模式。

## 何时使用

- 有人想了解自己的 MBTI 人格类型
- 已知自己或他人的 MBTI 类型，想要深入了解
- 分析两个人格类型之间的匹配度和相处建议
- 团队建设中分析成员性格搭配
- 职业规划中参考人格特质

## 何时不适用

- 临床心理诊断（MBTI 不是诊断工具）
- 招聘筛选（不应仅凭 MBTI 做人事决策）
- 替代专业心理咨询
- 对儿童的人格评估（MBTI 适用于成年人）

## MBTI 分维度测试

本测试分为4个维度，每个维度5道题。每题选 A 或 B，选完后统计每个维度的偏好。

### 计分规则

每个维度5题中：
- 选同一端 **4-5题** → **强烈偏好**
- 选同一端 **3题** → **轻微偏好**

四个维度组合确定类型，如：I(强烈) N(轻微) T(强烈) J(轻微) = INTJ

### 维度一：E/I — 能量来源（外向 / 内向）

**1. 周末你更倾向于：**
- A) 约朋友聚会、参加活动，在外待一整天
- B) 在家看书、看电影或做自己感兴趣的事

**2. 在一个陌生的社交场合，你通常会：**
- A) 主动找人搭话，很快融入氛围
- B) 先观察一会儿，等别人来找你或找到合适的人再开口

**3. 做完一个重要项目后，你更想：**
- A) 和团队一起庆祝，分享喜悦
- B) 一个人安静地放松，消化这件事

**4. 关于你的朋友圈：**
- A) 朋友很多，经常有不同圈子的社交活动
- B) 几个深交的朋友就够了，不需要太多社交

**5. 长时间独处后，你的感受是：**
- A) 感到有些孤独，想找人聊聊天
- B) 觉得很舒服，甚至不想被打扰

> 计分：A = E（外向），B = I（内向）

### 维度二：S/N — 信息获取方式（感觉 / 直觉）

**1. 阅读一篇文章时，你更容易注意到：**
- A) 文中的具体数据、事实和细节
- B) 文章的核心观点和背后的深层含义

**2. 解决问题时，你更依赖：**
- A) 过去的经验和经过验证的方法
- B) 直觉和灵感，尝试新的可能性

**3. 向别人描述一件事情时，你倾向于：**
- A) 按时间顺序讲清楚具体经过
- B) 先说结论和意义，再补充关键细节

**4. 你对未来的态度是：**
- A) 关注脚下的路，先做好眼前的事
- B) 经常想象各种未来的可能性，思考长远方向

**5. 学习新东西时，你更喜欢：**
- A) 有明确的步骤和操作指南
- B) 先了解整体概念，再自由探索

> 计分：A = S（感觉），B = N（直觉）

### 维度三：T/F — 决策方式（思考 / 情感）

**1. 朋友向你倾诉烦恼时，你的第一反应是：**
- A) 分析问题出在哪里，帮他理清思路找到解决方案
- B) 先共情和安慰，让他感受到你的理解和支持

**2. 做一个重要决定时，你更看重：**
- A) 逻辑分析和客观事实，即使结论不太讨喜
- B) 这个决定对相关的人的影响和感受

**3. 团队讨论中意见不一致时，你倾向于：**
- A) 坚持最有逻辑的方案，即使需要说服他人
- B) 寻找大家都能接受的折中方案，维护团队和谐

**4. 评价一个人的工作时，你更看重：**
- A) 结果和效率，用客观数据说话
- B) 过程和努力，考虑他付出的心血

**5. 当别人批评你时：**
- A) 先判断批评是否有道理，有则改之
- B) 先会有情绪反应，然后才理性分析

> 计分：A = T（思考），B = F（情感）

### 维度四：J/P — 生活方式（判断 / 知觉）

**1. 旅行前你的准备风格是：**
- A) 提前规划好详细的行程表，包括住宿和景点
- B) 只定大方向，到了当地再看心情安排

**2. 对待截止日期：**
- A) 通常提前完成，留出缓冲时间
- B) 经常在截止日期前冲刺，压力下效率最高

**3. 你的工作桌面 / 房间通常是：**
- A) 井井有条，东西都有固定位置
- B) 看起来有点乱，但我能找到需要的东西

**4. 做决定时你倾向于：**
- A) 尽快做出决定，然后按计划执行
- B) 保持开放，多收集信息，不到最后不急于定论

**5. 面对变化：**
- A) 更喜欢稳定和可预测的节奏
- B) 觉得变化带来新鲜感，不喜欢一成不变

> 计分：A = J（判断），B = P（知觉）
