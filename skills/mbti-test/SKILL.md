# MBTI 人格测试与类型分析

## 概述

基于 Myers-Briggs Type Indicator 的完整人格测试和分析工具。包含分维度测试、16种人格类型详细解读、以及双类型匹配度分析三大功能。

**核心框架：** 四个维度（E/I 外向/内向、S/N 感觉/直觉、T/F 思考/情感、J/P 判断/知觉）组合出16种人格类型，每种类型有独特的认知功能栈和行为模式。

## 何时使用

- 有人想了解自己的 MBTI 人格类型
- 已知自己或他人的 MBTI 类型，想要深入了解
- 分析两个人格类型之间的匹配度和相处建议
- 团队建设中分析成员性格搭配
- 职业规划中参考人格特质

## 何时不适用

- 临床心理诊断（MBTI 不是诊断工具）
- 招聘筛选（不应仅凭 MBTI 做人事决策）
- 替代专业心理咨询
- 对儿童的人格评估（MBTI 适用于成年人）
