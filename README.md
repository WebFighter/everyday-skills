# Everyday Skills

一套趣味心理测试与匹配分析工具集合，基于 [Claude Code Skills](https://docs.anthropic.com/en/docs/claude-code/skills) 构建。

## 包含工具

| 工具 | 说明 |
|------|------|
| **MBTI 人格测试** | 四维度测试 → 16 种人格类型解读 + 双类型匹配度分析 |
| **行为测试** | 36 道场景题 → 12 维度行为画像 → 16 种核心行为类型 |
| **心理年龄测试** | 20 道场景题 → 测出你的"真实年龄"，纯属娱乐 |
| **相亲匹配分析** | 多维度结构化匹配度评估，贴合中国相亲文化背景 |

## 使用方式

在 [Claude Code](https://docs.anthropic.com/en/docs/claude-code) 中加载对应 skill 即可使用。

## 项目结构

```
skills/          # 各 skill 定义
  mbti-test/
  behavior-test/
  mental-age/
  xiangqin-matching/
docs/            # 设计文档与实现计划
```
